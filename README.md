# 🏠 Airbnb Price Analysis Using Python

A data analysis project focused on exploring Airbnb listing prices, customer behavior, and pricing trends using Python and data visualization techniques.

---

## 📌 Project Overview

This project analyzes Airbnb datasets to identify:
- Price distribution
- Most expensive locations
- Room type trends
- Availability patterns
- Customer review insights

The project helps understand how different factors affect Airbnb pricing.

---

## 🚀 Features

✅ Data Cleaning  
✅ Exploratory Data Analysis (EDA)  
✅ Data Visualization  
✅ Price Trend Analysis  
✅ Location-Based Insights  
✅ Review Analysis  
✅ Statistical Insights  

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---
⚙️ Installation

Clone the repository:

git clone https://github.com/your-username/Airbnb-Price-Analysis.git

Install dependencies:

pip install -r requirements.txt

Run the project:

python analysis.py
